# GrantJudge

Score grant applications before humans waste review cycles.

Contract: [https://studio.genlayer.com/contracts?import-contract=0xebAA54f44dE5bF2B3FCEAD580d7e2C100d85084F](https://studio.genlayer.com/contracts?import-contract=0xebAA54f44dE5bF2B3FCEAD580d7e2C100d85084F)

## What it does

Paste the startup application and the fund thesis. The contract returns a ranked decision with the exact reasons the deal should move or pause.

## Why GenLayer

Real use case with fast business value.
Non-comparative equivalence keeps approval logic stable and easy to trust.
Simple UX: two text blocks and a decision.

## Contract mode

- Non-comparative equivalence

## Browser wallet flow

1. Open the app in a browser with the GenLayer Studio wallet available.
2. Paste the case input plus policy or rubric context.
3. Sign with the browser wallet and let the intelligent contract return the decision onchain.

## Local run

1. `pnpm install`
2. `pnpm dev`
3. Keep `.env.local` pointed at Studionet and the deployed contract address.
